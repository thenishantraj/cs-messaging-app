# Branch CS Messaging Web App

A modern, interactive customer service messaging application built with Streamlit for Branch's customer support team.

## Features

### Core Functionality
- **Multi-Agent Support**: Multiple agents can work simultaneously without authentication
- **Real-time Messaging**: Interactive chat interface with real-time updates
- **Message Queue**: Organized list of incoming customer messages
- **Customer Profiles**: Contextual customer information display
- **Canned Responses**: Quick reply templates for common inquiries

### Smart Features
- **Urgency Detection**: AI-powered scoring to flag urgent messages
- **Message Categorization**: Automatic classification of message types
- **Search & Filter**: Advanced search across messages and customers
- **Priority Management**: Visual indicators for message priority levels
- **Customer Context**: Integrated customer profiles and history

### UI/UX Highlights
- **Three-Column Layout**: Message queue, chat interface, and customer info side-by-side
- **Modern Design**: Slack-like chat bubbles with custom styling
- **Responsive Components**: Interactive elements with visual feedback
- **Real-time Updates**: Auto-refresh functionality for live updates
